使用说明:

### 下载

- 在TitleId里面输入要下载的id, 一行一个, 暂时只支持最新版
- 点击Download, 会按顺序下载, 对于已经下载的文件会判断大小自动跳过
- 下载的目录为NusData, 子目录名是TitleId
- 下载是单线程


### 导出 

- 在TitleId里面输入要下载的id, 一行一个, 暂时只支持最新版
- 点击Export, 会按顺序将下载列表导出到NusExport, 文件名中有文件大小信息


### 处理外部下载

- 点击Patch, 选择目录, 会检测所有文件是否有缺失, 大小是否正确, 并重命名为可安装的版本
- 这一步是同步操作, 界面可能会卡一会
